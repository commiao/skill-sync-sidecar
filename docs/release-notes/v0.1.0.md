# skill-sync-sidecar v0.1.0

Initial MVP release for WebDAV-backed private skill synchronization.

## Highlights

- Scans local skill roots and normalizes `SKILL.md` packages.
- Builds WebDAV-ready snapshots with per-skill archives and content hashes.
- Supports dry-run-first push, pull-cache, stage, apply, rollback, sync-status, sync-plan, sync-apply, sync-cycle, and sync-daemon flows.
- Blocks ambiguous peer-writer conflicts by default and materializes conflict review packages.
- Adds OpenClaw read-only reconcile gating for safe multi-device rollout.
- Includes current-device operational status checks and package smoke tests.

## Safety Boundary

- Does not sync credentials, API keys, tool databases, or runtime caches.
- Refuses accidental writes to the official `cc-switch-sync` prefix.
- Requires explicit `--yes` for remote uploads and local installs.
- Keeps OpenClaw writes out of the MVP release path.
