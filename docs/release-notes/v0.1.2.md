# skill-sync-sidecar v0.1.2

Release workflow hardening after the first GitHub Release run exposed a CI build dependency gap.

## Highlights

- Adds reusable `scripts/build-release-assets.sh` for local and CI release asset builds.
- Makes the GitHub Release workflow install build dependencies explicitly before building assets.
- Keeps the MVP sync behavior unchanged from `v0.1.0` and `v0.1.1`.
