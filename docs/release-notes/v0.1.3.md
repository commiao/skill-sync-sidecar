# skill-sync-sidecar v0.1.3

Patch release for Linux/OpenClaw runtime compatibility.

## Fixed

- Removed macOS-only `/private/tmp` staging assumptions from core library temp directories.
- `sync-apply`, conflict packaging, and tombstone packaging now use the platform default temporary directory.
- Added a regression test that prevents core temp staging from hardcoding `/private/tmp`.

## Validation

- Targeted sync-apply tempdir regression tests passed.
- `compileall` passed for `src` and `tests`.
